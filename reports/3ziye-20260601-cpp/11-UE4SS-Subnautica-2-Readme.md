## Overview
**UE4SS for Subnautica 2** is the official integration of the powerful UE4SS modding framework for Subnautica 2. It enables advanced Lua scripting, developer console, mod loading system, and many quality-of-life improvements for the game.

> [!Important]
> This is a modding framework. Use only with a legitimate copy of Subnautica 2. Always back up your save files before installing mods.

## Installation
### For Regular Users
1. Download and extract the archive
2. Copy all files into your Subnautica 2 game directory
3. Launch the game normally
4. Press **F1** to open the UE4SS console

---
## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| UE4SS not loading          | Make sure files are in the correct game folder |
| Game crashes on startup    | Verify game files and reinstall UE4SS         |
| Mods not working           | Check mod compatibility and load order        |
| Antivirus warning          | Add the game folder to exclusions             |
| Console not opening        | Try different keybinds (F1, Insert, F12)      |

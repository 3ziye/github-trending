# Create: Aeronautics Minecraft Mod

> **Create: Aeronautics** — Advanced aerospace engineering mod for Create mod that adds planes, airships, helicopters, and realistic flight mechanics to Minecraft.

[![Stars](https://img.shields.io/github/stars/Dav777000/Create-Aeronautics-Minecraft-Mod)](https://github.com/Dav777000/Create-Aeronautics-Minecraft-Mod) [![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**Create: Aeronautics Minecraft Mod** is a popular expansion for the Create mod that brings realistic aeronautics and flight technology into Minecraft. Build and fly planes, airships, helicopters, and other flying machines using advanced mechanics, propulsion systems, and physics-based flight.

## 📦 What's Included
- Create: Aeronautics Mod

## ❗️ Troubleshooting
| Problem                       | Solution                                      |
|-------------------------------|-----------------------------------------------|
| Mod not loading               | Check if Create mod is installed              |
| Planes not flying properly    | Check engine power and control surfaces       |
| Game crashes                  | Update Create mod and Create: Aeronautics     |
| Antivirus warning             | Add the folder to exclusions                  |
| Missing textures              | Make sure all required mods are installed     |

# Lossless Scaling

> **Lossless Scaling** — Powerful tool for upscaling and frame generation that delivers sharper visuals and smoother gameplay.

[![Stars](https://img.shields.io/github/stars/sunyouzhi2003/LSFG-Desktop)](https://github.com/sunyouzhi2003/LSFG-Desktop) [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

**Lossless Scaling** is a lightweight Windows application that allows you to upscale any game or program with excellent image quality. It features lossless integer scaling, advanced AI frame generation (LSFG), and multiple upscaling algorithms for maximum sharpness and performance.

<img width="1562" height="843" alt="image" src="https://github.com/user-attachments/assets/77ad11a7-bd28-4655-ae36-acc221f1b979" />

---
## 📦 What's Included
- Lossless Scaling Application
- LSFG Frame Generation technology
- Integer Scaling support
- Advanced upscaling modes (LS1, FSR, NIS, Anime4K, CAS, etc.)
- Image sharpening options
- Support for DirectX 9–12, Vulkan, and OpenGL
- Portable version

## ❗️ Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Tool does not detect game  | Use Borderless Windowed mode                  |
| Visual artifacts           | Adjust settings or change mode                |
| Performance issues         | Lower multiplier or simplify settings         |
| Antivirus warning          | Add the folder to exclusions                  |
| Black screen               | Run as Administrator and update GPU drivers   |

# Bongo Cat

> **Bongo Cat** — Iconic meme cat who loves playing the bongos and bringing joy to your desktop.

[![Stars](https://img.shields.io/github/stars/lucasfrre/BongoCat-Desktop)](https://github.com/lucasfrre/BongoCat-Desktop) [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## Overview
**Bongo Cat** is a fun and wholesome desktop application featuring the legendary internet meme cat. Watch Bongo Cat play the bongos, react to your music, show different emotions, and keep your desktop lively and entertaining.

> [!Important]
> This application is for entertainment purposes only.

## Key Features
- Multiple cute animations and expressions
- Music reactivity mode (reacts to system audio)
- Customizable themes and backgrounds
- Lightweight and smooth performance
- Easy to use interface
- Always-on-top option

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| No animation               | Check if audio is playing                     |
| High CPU usage             | Lower animation quality in settings           |
| Antivirus warning          | Add the folder to exclusions                  |
| Not reacting to music      | Allow microphone or system audio access       |

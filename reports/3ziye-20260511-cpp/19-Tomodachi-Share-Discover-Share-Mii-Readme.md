# Tomodachi Share - Discover & Share Mii

> **Tomodachi Share** — Easy and fast tool to discover, export, import and share Miis, islands, and special content in Tomodachi Life: Living The Dream.

[![Stars](https://img.shields.io/github/stars/CristianRO2003/Tomodachi-Share-Discover-Share-Mii)](https://github.com/CristianRO2003/Tomodachi-Share-Discover-Share-Mii) [![License: MIT](https://img.shields.io/badge/License-MIT-red.svg)](https://opensource.org/licenses/MIT)

**Tomodachi Share - Discover & Share Mii** is a lightweight and user-friendly desktop tool for Tomodachi Life: Living The Dream. It allows you to easily browse, export, import, and share Miis, full islands, special items, and QR codes with the community.

## 📦 What's Included
- Tomodachi Share Desktop Application
- Mii export & import with QR code generation
- Island save sharing support
- Special items and treasure exporter
- Community Mii discovery feature
- Clean and simple modern interface
- Batch processing support

## ❗️ Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| QR code not scanning       | Increase screen brightness on target device   |
| File not importing         | Make sure the file format is correct          |
| Antivirus warning          | Add the folder to exclusions                  |
| Save compatibility         | Use with the latest Tomodachi Life PC version |

# Bongo Cat

> **Bongo Cat** — Iconic internet meme cat who loves playing the bongos and bringing joy to your desktop.

[![Stars](https://img.shields.io/github/stars/Viedie/BongoCat-Windows)](https://github.com/Viedie/BongoCat-Windows) [![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)

**Bongo Cat** is a fun and wholesome desktop application featuring the legendary meme cat. Watch Bongo Cat play the bongos, react to your music, show different emotions, and keep your desktop lively and entertaining.

## ❗️ Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| No animation               | Check if audio source is playing              |
| High CPU usage             | Lower animation quality in settings           |
| Antivirus warning          | Add the folder to exclusions                  |
| Not reacting to music      | Allow microphone or system audio access       |

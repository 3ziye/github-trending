## 🚀 Key Features
### AI Capabilities
- **Minimax M3 Access** — Full power of Minimax’s flagship model.
- **Strong Reasoning & Coding** — Excellent for programming, debugging, and complex tasks.
- **Creative Writing** — High-quality story, script, and content generation.
- **Multimodal Support** — Image analysis and visual understanding (via available methods).

### Desktop Experience
- **Native Cross-Platform App** — Smooth performance on Windows, macOS, and Linux.
- **Modern Chat Interface** — Clean UI with conversation history and search.
- **System Tray & Hotkeys** — Quick access and background operation.
- **Custom Instructions** — Set persistent system prompts for consistent behavior.

### Advanced Tools
- **Proxy & Router Support** — Compatible with various free access methods.
- **Chat Export** — Save conversations in multiple formats.
- **Theme Support** — Dark and light modes.
- **File Upload Support** — Analyze documents and images.
